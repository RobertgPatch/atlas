# Spec 004 - Partnership Management

## Goal
Create CRM-style operational views for entities and partnerships.

## Scope
- partnership directory
- partnership detail
- entity detail
- FMV snapshots

## Functional Requirements
- browse partnerships in list and optional card views
- open partnership detail
- open entity detail
- display K-1 history, FMV snapshots, expected distributions

## Acceptance Criteria
- entity-to-partnership relationship is visible
- views are reusable and consistent
- FMV snapshots can be created and viewed
