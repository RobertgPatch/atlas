# Spec 005 - Dashboards

## Goal
Deliver shared default dashboards for executive and operational use.

## Scope
- Main Dashboard
- K-1 Processing Dashboard
- widget framework

## Functional Requirements
- main dashboard shows executive metrics
- K-1 processing dashboard shows workflow metrics
- FMV bubble chart uses latest partnership FMV snapshot
- widgets render from stored config

## Acceptance Criteria
- both dashboards exist
- widgets render correctly
- shared dashboard model works
