# Spec 007 - Admin Management

## Goal
Provide admin surfaces for managing users and roles.

## Scope
- user management
- invite flow
- deactivate/reactivate
- role assignment

## Functional Requirements
- admin can view users
- admin can create/invite users
- admin can assign/remove roles
- admin can deactivate users

## Acceptance Criteria
- non-admin access is blocked
- audit events are created for all admin actions
